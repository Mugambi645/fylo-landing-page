# fylo-landing-page

A fylo landing page created using react and tailwind css.

## Installation
Node.js
Npm

```bash
git clone https://github.com/Mugambi645/fylo-landing-page
```

## Usage

```
cd fylo-landing-page
npm install
npm start
```
